# 更新日志

[English](CHANGELOG.md) | [简体中文](CHANGELOG.zh-CN.md)

## 0.9.0 - 2026-07-19

- 应用退出时等待所有受管 SSH Runner 完成 ControlMaster 关闭并退出；有界 TERM/KILL 兜底只作用于应用自己创建的 SSH 子进程。
- 使用结构化 `lsof -F` 输出，为每条规则自动检查本地配置端口。
- 显示每个本地配置端口对应的监听进程名和 PID，代理规则会分别检查 SOCKS 与 HTTP 端口。
- 为外部监听进程增加需要明确确认的 SIGTERM 操作，同时禁止终止 PID 1 或 SSH Proxy Tray 自身。
- 按规则隔离端口检查状态，切换规则时不会显示其他 profile 的旧结果。

## 0.8.1 - 2026-07-19

- 增加应用内语言弹出面板，支持“跟随系统 / 简体中文 / English”，整个窗口无需重启即可立即切换。
- 将笼统的远程转发失败改为本地化说明，明确提示远程端口可能已被其他 SSH 会话占用，或被服务器策略拒绝。
- 正常断开时先显式关闭 OpenSSH ControlMaster，再以进程终止兜底，降低遗留远程转发的概率。
- 将 `CFBundleInfoDictionaryVersion` 保持为 Apple 标准的 `6.0`，与应用版本独立管理。

## 0.8.0 - 2026-07-19

- 将窗口壳层重做为消费级桌面工作区，不再使用传统设置表单的整体观感。
- 增加浅色导航侧栏、悬浮规则项、品牌图标和更克制的底部工具区。
- 移除系统默认分栏、分段控件和输入框外观，改用自定义内容顶栏、模式选择器、访问范围选择器、认证菜单和柔和输入面板。
- 内容延伸到标题栏下方，形成更干净的 macOS 窗口；隧道运行时锁定的字段仍保持可读。

## 0.7.0 - 2026-07-19

- 重做 macOS 管理窗口视觉层：深色导航侧栏、明亮工作区、更清晰的状态胶囊和分组配置面板。
- 强化连接路径、规则类型、启用状态以及连接/断开主操作的视觉层级。
- 增大默认窗口尺寸，并统一使用圆角输入控件，让编辑过程更舒展、更易扫描。
- 保持代理、转发、本地化、凭证和远程监听行为不变。

## 0.6.0 - 2026-07-18

- 重新设计管理窗口，突出规则状态、连接路径、紧凑配置分区和主要操作。
- 将远程转发的原始绑定地址输入改为“仅 SSH 服务器 / 外部设备 / 自定义地址”访问范围。
- 通过当前已认证的 OpenSSH ControlMaster 会话检查服务器实际远程监听。
- 区分已确认 loopback、已确认外部监听、监听缺失、正在检查和无法检查等状态。
- 为支持的 systemd/OpenSSH 服务器增加需要用户明确确认的 `GatewayPorts clientspecified` 配置，包含备份、`sshd -t`、reload、失败回滚、原地刷新转发和复验。
- 记录 ad-hoc 构建的 Gatekeeper 提示、校验和验证、可信来源 quarantine 临时处理方法，以及 Developer ID/notarization 长期方案。
- 增加对 OpenAI ChatGPT 全程开发协助和 ZhaoYang1 算力支持的致谢。
- 保持完整英文与简体中文界面、README、Security 和 Changelog。

## 0.5.0 - 2026-07-18

- 一条代理规则可以通过同一个 `ssh -D` 连接同时提供 SOCKS5 与可选 HTTP/HTTPS 端点。
- 增加只监听 loopback 的本机 HTTP-to-SOCKS 适配器，支持 HTTPS CONNECT 和普通 HTTP 代理请求。
- 增加独立的 SOCKS/HTTP 端口配置、冲突校验、端点摘要和复制操作。
- 保持向后兼容：旧 profile 在用户明确启用前仍只提供 SOCKS；新建代理规则默认启用两个端点。
- 增加有界 HTTP 请求头与握手、代理凭证头过滤、解析/编码测试，以及中英文界面和安全说明。

## 0.4.0 - 2026-07-18

- 为可用状态与连接状态、规则类型、认证、转发和 SSH 高级参数增加中英文就地帮助。
- 用具体使用场景和方向示例解释 SOCKS、本地转发和远程转发的数据流。
- 为绑定地址、监听端口、目标字段、ProxyJump、压缩、超时和保活参数增加字段级悬停提示。
- 规则连接后配置字段锁定时，帮助按钮仍然可以使用。

## 0.3.0 - 2026-07-18

- 增加完整英文与简体中文应用本地化，自动跟随 macOS 首选语言。
- 本地化连接状态、操作按钮、配置表单、校验和运行错误，不改变已保存 profile 的字段值。
- 增加完整中英文 README，以及中文 Security 和 Changelog 文档。
- 使用双语仓库描述、徽章、检索 topics 和双语 Release Notes 提升 GitHub 可发现性。

## 0.2.0 - 2026-07-18

- 用状态栏图标直接打开的独立连接管理窗口替代菜单栏 popover。
- 支持多条规则并发运行，每条规则拥有独立启用状态和运行状态。
- 为每条规则增加手工连接和断开操作。
- 增加 SOCKS 代理、本地转发和远程转发三种规则类型。
- 支持从 `~/.ssh/config` 导入具体 Host alias。
- 增加 SSH certificate、ProxyJump、压缩、超时和 keepalive 设置。
- 新 profile 默认从 18080 选择端口，避免常见代理端口冲突。
- SSH 启动前检测本地端口占用，并显示明确错误。

## 0.1.0 - 2026-07-17

- 增加原生 macOS 状态栏应用。
- 支持 SSH config alias、私钥和可选钥匙串密码认证。
- 支持 SOCKS5 动态转发和远端 HTTP/HTTPS 代理端口转发。
- 支持自动连接和登录时启动。
- 增加 profile CLI、本地应用打包和测试。
